# Synthetic Data Field Reference

Complete documentation of all asset types and fields generated by `test-synthetic-data.js`

---

## Overview

The synthetic data generator creates realistic test data for three source types:

1. **VMware vCenter** - Virtual machines
2. **IP Fabric** - Network devices
3. **Snipe-IT** - IT assets

All data uses **faker.js** to generate random but realistic values.

---

## 1. VMware vCenter Virtual Machines

### Asset Type
**Virtual Machines (VMs)** - Server and workstation virtualization

### Generated Fields

#### Root Level Fields

| Field | Type | Example | Description |
|-------|------|---------|-------------|
| `id` | String | `"vm-a1b2c3d4"` | Unique VM identifier (UUID format) |
| `name` | String | `"Keyboard-VM-1"` | VM display name (product + VM + number) |

#### Hardware Section (`hardware`)

| Field | Type | Example | Description |
|-------|------|---------|-------------|
| `hardware.numCPU` | Number | `4` | Number of virtual CPUs (2, 4, 8, or 16) |
| `hardware.memoryMB` | Number | `16384` | Memory in MB (4096, 8192, 16384, or 32768) |
| `hardware.numCoresPerSocket` | Number | `2` | Cores per socket (always 2) |
| `hardware.version` | String | `"vmx-17"` | VM hardware version |

#### Guest OS Section (`guest`)

| Field | Type | Example | Description |
|-------|------|---------|-------------|
| `guest.guestFullName` | String | `"Microsoft Windows Server 2019 (64-bit)"` | Full OS name |
| `guest.guestId` | String | `"windows9Server64Guest"` | VMware guest OS identifier |
| `guest.hostName` | String | `"server01.company.com"` | FQDN hostname |
| `guest.ipAddress` | String | `"192.168.1.10"` | Primary IP address |

**Possible OS Values**:
- Microsoft Windows Server 2019 (64-bit)
- Microsoft Windows Server 2016 (64-bit)
- Red Hat Enterprise Linux 8 (64-bit)
- Ubuntu Linux (64-bit)
- CentOS 7 (64-bit)

#### Runtime Section (`runtime`)

| Field | Type | Example | Description |
|-------|------|---------|-------------|
| `runtime.powerState` | String | `"poweredOn"` | Power state (75% on, 25% off) |
| `runtime.bootTime` | String | `"2025-11-01T10:00:00Z"` | Last boot time (ISO 8601) |
| `runtime.maxCpuUsage` | Number | `12000` | Max CPU usage in MHz (4000-16000) |
| `runtime.maxMemoryUsage` | Number | `16384` | Max memory usage in MB (4096-32768) |

#### Config Section (`config`)

| Field | Type | Example | Description |
|-------|------|---------|-------------|
| `config.uuid` | String | `"a1b2c3d4-..."` | BIOS UUID |
| `config.instanceUuid` | String | `"e5f6g7h8-..."` | Instance UUID |
| `config.guestId` | String | `"windows9Server64Guest"` | Guest OS ID |
| `config.annotation` | String | `"Production web server"` | VM notes/description |
| `config.template` | Boolean | `false` | Is template (always false) |

#### Summary Section (`summary`)

Nested structure with config, runtime, and guest info:

| Field | Type | Example | Description |
|-------|------|---------|-------------|
| `summary.config.name` | String | `"WebServer-VM-1"` | VM name |
| `summary.config.vmPathName` | String | `"[datastore1] VM-1/VM-1.vmx"` | Path to VM files |
| `summary.config.memorySizeMB` | Number | `8192` | Memory size (4096, 8192, 16384) |
| `summary.config.numCpu` | Number | `4` | CPU count (2, 4, 8) |
| `summary.runtime.powerState` | String | `"poweredOn"` | Current power state |
| `summary.runtime.host` | String | `"host-5"` | ESXi host (host-1 to host-10) |
| `summary.guest.ipAddress` | String | `"10.0.1.50"` | Guest IP address |
| `summary.guest.hostName` | String | `"web01.domain.com"` | Guest hostname |

## Data Realism

### faker.js Methods Used

- **Names**: `faker.commerce.product()`, `faker.company.companyName()`
- **Network**: `faker.internet.ip()`, `faker.internet.domainName()`, `faker.internet.mac()`
- **Identifiers**: `faker.datatype.uuid()`, `faker.random.alphaNumeric()`
- **Numbers**: `faker.datatype.number()`, `faker.random.arrayElement()`
- **Dates**: `faker.date.recent()`, `faker.date.past()`
- **Text**: `faker.lorem.sentence()`, `faker.lorem.sentences()`
- **Users**: `faker.name.findName()`, `faker.internet.userName()`, `faker.internet.email()`

### Randomization

- **Weighted values**: VMs are 75% powered on, 25% off
- **Optional fields**: Snipe-IT assets are 50% assigned, 50% unassigned
- **Realistic ranges**: CPU (2-16), Memory (4-32GB), Cost ($500-$5000)
- **Date ranges**: Boot times (last 30 days), purchases (last 3 years)
